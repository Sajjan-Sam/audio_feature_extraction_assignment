## About
This project is on **audio feature extraction** using Python and Librosa.  
We analyzed **9 audio signals** and extracted features from:

- **Time domain**
- **Frequency domain**
- **Time-frequency domain**

We also plotted:

- **Waveform**
- **Frequency Spectrum**
- **Spectrogram**
- **MFCC**

---

## Audio Files Used

### Librosa Example Files
- `libri1` → Speech
- `choice` → Music
- `trumpet` → Instrument

### Custom Files
- `Valve1.wav`
- `Music1.mp3`
- `Music2.mp3`
- `Speech1.mp3`
- `Speech2.mp3`
- `Gear1.wav`

---

## Features Extracted

### Time Domain
- Amplitude Mean
- Amplitude Standard Deviation
- RMS Energy
- Zero Crossing Rate
- Duration

### Frequency Domain
- Dominant Frequency
- Spectral Centroid
- Spectral Bandwidth
- Spectral Rolloff

### Time-Frequency Domain
- MFCC Mean
- MFCC Standard Deviation
- Chroma Mean
- Mel Spectrogram Mean
